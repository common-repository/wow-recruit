== Description ==
WoW Recruit provides an easy to use tool for world of warcraft guilds to show current recruitment needs on their website.


== Installation ==


1. Upload the `wow-recruit.php` plugin to your blogs `wp-content/plugins/` folder
2. Activate it using the plugin manager
3. Drag the Wow Recruit widget to the sidebar of your choice
